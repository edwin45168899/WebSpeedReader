# WebSpeedReader 開發任務清單 (TODO List)

這個清單用於追蹤 `WebSpeedReader` 的後續優化計畫，確保開發進度條理清晰。

## 🟢 已完成 (Done)
- [x] **語音朗讀 (TTS)**: 新增朗讀按鈕，可將 AI 總結內容透過瀏覽器語音讀出，支援多語言切換與播放/暫停。
- [x] **自定義總結風格**: 支援簡明 (Concise)、標準 (Normal) 或詳細 (Detailed) 三種模式切換。
- [-] **快捷鍵支援**: 因應 V3 環境限制與熱鍵衝突頻繁，暫時放棄快捷鍵實作。
- [x] **多語系翻譯優化**: 擴充支援日語、韓語、法語、德語、西班牙語。
- [x] **全域色彩聯動**: 允許使用者完全自定義「背景顏色」與「字體顏色」，按鈕外觀會自動適配。
- [x] **歷史紀錄管理**: 支援單筆刪除、清空紀錄與匯出 JSON 資料。
- [x] **多模型選擇**: 提供選單讓使用者切換不同的 Groq 模型。
- [x] **深色模式支援**: 根據系統設定或切換按鈕美化介面。
- [x] **閱讀量節省統計**: 顯示原字數 vs 總結字數，直觀展示效率提升。
- [x] **右鍵選單功能**: 選取網頁一段文字後，點擊右鍵即可「總結選取內容」。
- [x] **歷史紀錄 (History)**: 保存最近 10 次總結，支援離線查看。
- [x] **精確內容提取 (Smart Extraction)**: 整合 `Readability.js` (Mozilla) 以過濾網頁雜訊，提升總結品質。
- [x] **Markdown 渲染**: 引入 `marked.min.js` 將 AI 回傳的總結轉化為美觀的 HTML。
- [x] **一鍵複製**: 新增複製原始 Markdown 內容的功能並提供 UI 回饋.
- [x] **API Key 安全與佈局**: 將 Key 輸入移至頂部並改為密碼格式，提供尾碼校驗。
- [x] **基礎多語言**: 支援繁體中文與英文切換。
- [x] **專案規範文件**: 建立 `GEMINI.md` 與 `CHANGELOG.md`。
- [x] **載入狀態美化**: 加入 Loading 動畫與打字機光標效果，提升互動感。
- [x] **語音朗讀 (TTS)**: 支援將總結內容朗讀出來，並可自動切換語言。
